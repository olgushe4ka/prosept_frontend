## Build and Run Instructions

### Prerequisites

- Node.js (v14 or later)
- npm (v7 or later)
- TypeScript (v5.2.2)

### Installation

git clone https://github.com/olgushe4ka/prospect_frontend
cd prospect_frontend
npm ci

### Development

npm run start
Visit http://localhost:5173 in your browser.

### Production Build

npm run build

### Technology Stack

React
TypeScript
Axios
